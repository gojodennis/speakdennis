## Design Context

### Users
Language learners practicing speech, engaging with audio scenarios and receiving feedback. They need an environment that feels welcoming and minimizes the anxiety often associated with learning a new language or speaking out loud.

### Brand Personality
Cozy, premium, inviting. The tone is encouraging, refined, and playful yet polished.

### Aesthetic Direction
A "cozy pastel light mode" aesthetic infused with high-end, immersive structural details. 
- **Palette:** Soft pastels (creamy whites, soft lavenders, gentle blues/greens) serving as the cozy baseline.
- **Structure:** Taking inspiration from premium travel apps—large full-bleed visual elements, smooth glassy overlaps, floating pill-based navigation, and highly refined typographic alignments.
- **Vibe:** It should feel like a high-end native mobile application that happens to live on the web (PWA-style mobile-first layout).

### Design Principles
1. **Mobile-First Purity:** The interface is built strictly for mobile proportions, using a centralized device-width container on larger screens.
2. **Soft Depth over Flatness:** Use incredibly soft, diffused, multi-layered shadows to separate elements, avoiding harsh lines and borders.
3. **Immersive Framing:** Utilize smooth gradients and edge-to-edge structural blocks that make content feel like windows into a world.
4. **Deliberate Typography:** High contrast display typography mapped against gentle pastels.
